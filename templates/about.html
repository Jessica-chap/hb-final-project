Nanjo: Fitness Tracker
  • Users can create an individual account
  • Track their bodyweight
  • Create workouts with customizable measurements/inputs
  • Save these workouts to use in the future

This project is the culmination of what I learned during the first 10 weeks of my Hackbright, Perficient sponsored, full stack engineering boot camp. In the future I plan to expand the app as I learn new technologies and gain more experience in the field.

Technologies used:
  • Languages- Python,Javascript, Json, HTML, CSS
  • Frameworks/Libraries- Flask, Jinja2, Pytest, Bootstrap, jQuery, AJAX, SQL, SQLAlchemy, PostgreSQL, Git, GitHub
